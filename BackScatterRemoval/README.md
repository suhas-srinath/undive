# Backscatter Removal
For a help Menu
use ```python <filename> -h```
## Per Video
```bash
python bsr_per_video.py --video-path <video_frames> --depthmap-path <depthmaps> --output-path <bsr_images>
```

## For multiple Videos 
```bash
python bsr.py --video-path <videos> --depthmap-path <depthmaps> --output-path <bsr_images>
```
